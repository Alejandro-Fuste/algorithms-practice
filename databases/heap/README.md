# Max Heap

## Heap

Heap: a type of tree-like data structure that uses an array to store data

## Max Heap

Max Heap: is where the parent is always greater than any of its children

## Index Structure

| Node        | Index           |
| ----------- | --------------- |
| (itself)    | i               |
| Parent      | Math.floor(i/2) |
| Left Child  | i \* 2          |
| Right Child | i \* 2 + 1      |
